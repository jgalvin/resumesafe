# ResumeSafe — beta landing page

Static placeholder for the ResumeSafe private beta. Single `index.html`, hosted on GitHub Pages.

Beta signups insert into the ResumeSafe Supabase project (`signups` table, anon insert-only RLS). The anon key in the page is public by design.

Note: `resumesafe.com` still serves the legacy AWS site, so this is published on the github.io URL until a domain/subdomain is pointed at it.
